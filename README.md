# BadgingSystem
A Modular badge-in system that uses a device to scan a QR code. Phones allow users to create and use their badge, and admin allows us to monitor our device and leaderboard.

# Tech Stack
Frontend:
- iOS/Android - Expo (React Native)
- Tablet - SwiftUI
- Admin - NextJS
Backend: NestJS
Database: MongoDB (Firebase)
Authentication: Clerk
Hosting: Vercel
